核心功能：

可扩展架构 — 在 src/renderer/src/tools/registry.tsx 中添加新工具即可自动出现在首页，路由自动注册
音频剪辑 — 基于 wavesurfer.js 的波形显示，支持拖拽选区裁剪、实时播放预览、导出 WAV
文件操作 — 支持文件对话框打开和拖拽导入
CI/CD — 推送 v* 标签自动在 GitHub Actions 中构建 Windows/Mac/Linux 安装包并发布到 Release
使用命令：

pnpm dev — 启动开发模式
pnpm build — 构建应用
pnpm build:win — 打包 Windows 安装包
pnpm build:mac — 打包 Mac 安装包
pnpm build:linux — 打包 Linux 安装包
pnpm lint — ESLint 检查
扩展新工具示例 — 只需在 registry.tsx 中新增一条：

```tsx
{
  id: 'new-tool',
  name: '新工具',
  description: '工具描述',
  icon: <SomeIcon />,
  path: '/tools/new-tool',
  component: lazy(() => import('../pages/tools/NewTool'))
}
```
